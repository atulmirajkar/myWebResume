Implementation of Manufacturing Process Planner for Nissan (Automobile Industry Japan)

Work Experience - Geometric Ltd - C++, C, Java
Geometric Ltd provides PLM (Product Lifecycle Management) Solutions to Manufacturing Sector

Project Description

The client Nissan (a major automobile company in Japan) has a concept of Engineering BOM where in their Engineering department creates their CAD design and store them in an Engineering BOM (EBOM) structure. In order to create Manufacturing plan for these designed items in early stage of vehicle design, The Engineering Data needs to be mapped to a Planning BOM (PBOM) and via Planning BOM to the Bill of Process (BOP).

Responsibilities

  • Server Side Customization C/C++ mainly related to Manufacturing Process Planner in Teamcenter (A Siemens PLM product).
  • Rich Client UI Java Customization according to customer requirement.
  • Help in creation of Functional Specification according to requirement understanding.
  • Creating Proof of Concepts for requirements from time to time.
